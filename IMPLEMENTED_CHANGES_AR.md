# التعديلات المنفذة

- إضافة النشاط التفصيلي وتاريخ التأسيس وعمر الشركة عند التسجيل وفي طلب التمويل.
- حفظ السجل التجاري وبيانات الشركة داخل ملف الطلب الظاهر للبنك.
- إضافة صفحة متابعة التمويل للشركة تعرض الموافقة المشروطة، الدفعات، جدول الأقساط، طلبات البنك وإجراءات التحصيل.
- إضافة رفع القوائم المالية الشهرية أو كشف الحساب من داخل ملف التمويل وإرسالها مباشرة للبنك.

# بوابة الشركات — النبض المالي

هذه النسخة مخصصة للشركات فقط: التسجيل، رفع البيانات، تقديم طلب التمويل، الاستعلام، ومتابعة الضمانات بعد موافقة البنك.

## التشغيل

```bash
cd financial-pulse-company-portal
npm install
cp .env.example .env.local
npm run dev
```

ثم افتحي: `http://localhost:3000`

## الربط مع بوابة البنك

ضعي **نفس قيمة `DATABASE_URL`** داخل `.env.local` في بوابة الشركات وبوابة البنك. يجب تنفيذ ملف `database/schema.sql` مرة واحدة على قاعدة PostgreSQL قبل أول تشغيل.

> لا تعتمدي على التخزين المحلي `data/db.json` عند تشغيل البوابتين كمشروعين منفصلين؛ لأنه سيكون ملفًا مختلفًا في كل مشروع ولن تنتقل الطلبات بينهما.

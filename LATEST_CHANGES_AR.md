# آخر تعديلات بوابة الشركات

## الواجهة وتجربة المستخدم
- إزالة زر Next.js الأسود الدائري `N` من وضع التطوير.
- إعادة تصميم الصفحة الرئيسية بأسلوب خفيف يعتمد على الخطوط والمساحات الهادئة بدل كثرة المربعات.
- عرض رحلة التمويل في مسار واضح من رفع القوائم حتى الصرف والمتابعة.
- إضافة قسم مرتب للمستندات المطلوبة والأسئلة الشائعة.
- إزالة واجهات تسجيل/إنشاء حساب الشركة ولوحة الشركة من تجربة المستخدم، وتحويل المتابعة إلى صفحة الاستعلام الآمنة برقم المعاملة وOTP.

## طلب التمويل والمستندات
- القوائم المالية تقبل Excel أو CSV أو PDF النصي.
- كشف الحساب البنكي لآخر 6 أشهر: إلزامي.
- السجل التجاري: إلزامي.
- هوية المفوض بالتوقيع: إلزامي.
- خطة استخدام مبلغ التمويل: اختيارية.
- الملاحظات الإضافية: اختيارية.
- دراسة الجدوى تظهر فقط عند اختيار إنشاء شركة جديدة/تمويل تأسيس.
- مستندات الضمان تُطلب عند الحاجة وبعد الموافقة المبدئية أو المشروطة.

## المتابعة بعد تقديم الطلب
- تعرض صفحة الاستعلام قرار البنك والشروط والضمانات وحالة الصرف.
- تعرض جدول الأقساط والسداد.
- تعرض المتابعة الشهرية والإنذار المبكر.
- يمكن رفع القوائم الشهرية أو كشف الحساب من صفحة الاستعلام بعد التحقق برمز OTP.

## ملاحظة PDF
يُقبل PDF الذي يحتوي على نص قابل للاستخراج. ملفات PDF المصورة بالماسح الضوئي تحتاج OCR قبل رفعها أو تحفظ كـ Excel/CSV.

## التشغيل
```bash
npm install
cp .env.example .env.local
npm run dev
```

تعمل البوابة افتراضيًا على:
`http://localhost:3000`

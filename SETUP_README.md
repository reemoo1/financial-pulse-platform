# تشغيل بوابة الشركات — النبض المالي

هذه النسخة مخصصة للشركات فقط، وتشمل التسجيل، رفع البيانات المالية، تقديم طلب التمويل، الاستعلام، ومتابعة الضمانات بعد قرار البنك.

## التشغيل المحلي

```bash
cd financial-pulse-company-portal
npm install
cp .env.example .env.local
npm run dev
```

ثم افتحي:

```text
http://localhost:3000
```

## ربطها ببوابة البنك

1. أنشئي قاعدة PostgreSQL واحدة.
2. نفذي ملف `database/schema.sql` على القاعدة مرة واحدة.
3. ضعي رابط القاعدة نفسه في ملف `.env.local` للمشروعين:

```env
DATABASE_URL=postgresql://...
```

4. ضعي مفتاح جلسة لا يقل عن 32 حرفًا:

```env
SESSION_SECRET=ضع-هنا-مفتاحا-طويلا-وآمنا-لا-يقل-عن-32-حرفا
```

> عند ترك `DATABASE_URL` فارغًا سيستخدم كل مشروع ملفًا محليًا منفصلًا، وبالتالي لن تنتقل الطلبات والقرارات بين البوابتين.
